# AGENTS.md

## Тип проекта
- Приложение Windows Forms (.NET Framework 4.6.2) на C#
- Использует компоненты DevExpress (v24.2)
- Содержит функциональность VPN и RDP подключения

## Сборка и версионирование
- Версия автоматически генерируется с помощью GenerateVersion.ps1 во время предварительной сборки
- Формат версии: 1.ГГ.ММДД.РР (ГГ=год, ММДД=месяц+день, РР=релиз)
- Сборка запускается: `GenerateVersion.ps1` перед каждой сборкой
- Путь вывода для релиза: D:\Release\VaR\

## Ключевые компоненты
- Основная точка входа: Program.cs → LoginForm.cs → ConnectionForm.cs
- Функциональность VPN: VPNManager.cs + tunnel.dll + wireguard.dll
- Функциональность RDP: COM ссылки на MSTSCLib/AxMSTSCLib

## Зависимости
- Внешние DLL: tunnel.dll, wireguard.dll (копируются в выходную директорию)
- Пакеты NuGet: CredentialManagement, DotRas, Newtonsoft.Json, NLog, RestSharp
- Ссылки на проекты: ConfigFile, ConnectLIbrary, Logs, OSVersionInfoDLL

## Тестирование и запуск
- Тестовая среда не обнаружена — вероятно, ручное тестирование
- Запуск через ShortCut.bat (создает ярлык на рабочем столе VaR.exe)
- Требуется .NET Framework 4.6.2 и Windows

## Важные замечания
- Не изменяйте AssemblyVersion/AssemblyFileVersion вручную — пусть GenerateVersion.ps1 обрабатывает это
- Всегда запускайте `GenerateVersion.ps1` перед сборкой для правильного версионирования
- Приложение использует нативные DLL (tunnel.dll, wireguard.dll), которые должны присутствовать в выходной директории
- Функциональность RDP зависит от компонентов удаленного рабочего стола Windows
- Конфигурация хранится в app.config и Properties.Settings

## Среда разработки
- Visual Studio с .NET Framework 4.6.2
- Требуется Windows OS (не кроссплатформенное приложение)
- Нет CI/CD пайплайна — вероятно, ручное развертывание

## Архитектурные заметки
- Единый исполняемый файл с встроенным интерфейсом
- Нет отдельных серверных компонентов
- Конфигурация хранится в app.config и пользовательских настройках
- Использует Windows-специфичные API для VPN и RDP

## Критические зависимости
- tunnel.dll — драйвер VPN туннеля
- wireguard.dll — реализация WireGuard
- DotRas — обертка для API набора dial-up соединений
- CredentialManagement — хранение учетных данных Windows
- NLog — логирование
- RestSharp — HTTP клиент

## Процесс сборки
1. Выполнить GenerateVersion.ps1 (предварительное событие сборки)
2. Скомпилировать C# код с помощью MSBuild
3. Скопировать нативные DLL в выходную директорию
4. Скопировать ShortCut.bat в выходную директорию

## Требования к запуску
- Windows 7 или выше
- .NET Framework 4.6.2
- Клиент RDP установлен
- Привилегии администратора для функциональности VPN
- Доступ к сетевым ресурсам

## Развертывание
- Ручное копирование папки D:\Release\VaR\ на целевую машину
- Запуск ShortCut.bat для создания ярлыка на рабочем столе
- Нет установщика

## Устранение неполадок
- Если VPN не работает: проверьте наличие tunnel.dll и wireguard.dll
- Если RDP не работает: убедитесь, что клиент RDP Windows доступен
- Если версия не обновляется: проверьте выполнение GenerateVersion.ps1
- Если настройки теряются: проверьте app.config и пользовательские настройки
- Если приложение крашится: проверьте Просмотр событий Windows и app.log
